# Portfólio — Victor Boens (Fluent UI)
Site com layout em Fluent UI (React) publicado no GitHub Pages.

- Live: https://victorboens-qa.github.io/victorboens-qa
- Stack: Vite, React, Fluent UI v9

## Scripts
- `npm run dev` — desenvolvimento
- `npm run build` — build
- `npm run preview` — preview local
- `npm run deploy` — publica `dist` no GitHub Pages
